# Changelog

All notable changes to AWSRT will be documented in this file.

The project uses a versioned research-software workflow in which frozen releases
are preserved for reproducibility and ongoing development continues on later
branches.

## [v0.2-dev]
### Added
- Initial post-v0.1 development branch for continued platform work.
- Repository preparation for first public release.
- Updated `.gitignore` for public/open-source publication hygiene.

### Changed
- Continued refinement of documentation and reproducibility materials.
- Ongoing development is now separated from the frozen `v0.1` release.

## [v0.1] - 2026-03-31
### Added
- Frozen AWSRT v0.1 research software release associated with the thesis-results version of the platform.
- Preserved manifests and figures supporting the frozen study state.
- Reproducibility and versioning notes for the frozen release.

### Included
- Physical wildfire simulation and historical replay support.
- Epistemic belief evolution and entropy-oriented analysis.
- Operational sensing experiments with multiple policy families.
- Impairment studies involving delay, noise, and loss.
- Analysis and figure-generation workflows for baseline, wedge, and regime studies.

### Notes
- `v0.1` is intended as a meaningful frozen research release rather than the final form of AWSRT.
- Active development continues in `v0.2-dev`.
